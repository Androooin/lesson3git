# Инструкция для работы с Markdown

## Выделение текста

Чтобы выдлить текст курсивом необходимо обрамить его звездочками (*). Например, *вот так*.

Чтобы Выделить текст полужирным, необходимо обрамить его двойными звездочками (**).
Например, **вот так**.
